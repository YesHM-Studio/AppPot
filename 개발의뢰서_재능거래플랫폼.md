# 재능거래 플랫폼 개발 의뢰서

안녕하세요.

크몽과 유사한 재능거래 플랫폼 제작을 의뢰하고자 합니다.

---

## 📌 목표

- **의뢰자**가 프로젝트를 등록하고,
- **판매자**가 견적을 제안하며,
- 플랫폼 내에서 **결제 및 채팅**이 가능한 구조입니다.

의뢰자와 판매자를 연결하는 웹 기반 반응형 플랫폼입니다.

---

## 🔧 필수 기능

### ① 회원 시스템
- **일반회원 (의뢰자)**: 프로젝트 등록·의뢰 가능
- **판매회원 (서비스 제공자)**: 견적 제안, 포트폴리오 등록
- **관리자 페이지**: 회원·서비스·신고·정산 관리

### ② 의뢰 등록 기능
- 제목 입력
- 카테고리 선택
- 예산 입력
- 마감일 설정
- 파일 첨부 가능

### ③ 판매자 기능
- 프로필 생성
- 포트폴리오 업로드
- 견적 제안 기능
- 1:1 채팅 기능

### ④ 결제 시스템
- PG 연동 (토스 or 아임포트)
- 에스크로 방식
- 완료 후 정산

### ⑤ 관리자 기능
- 회원 관리
- 신고 관리
- 정산 승인

---

## 💻 기술 스택

- 기술 스택 제안 부탁드립니다.
- **React** 기반 웹앱 희망합니다.
- Node.js 백엔드 가능 여부 검토 부탁드립니다.
- DB: **MySQL** 또는 **Firebase** 고려 중입니다.
- **AWS 배포** 포함 견적 요청드립니다.

---

## 🎨 디자인 방향

- **미니멀하고 세련된 UI**
- **미리캔버스 + 크몽** 느낌을 섞은 구조
- **모바일 UX 최적화 필수** (반응형 필수)
- **메인 컬러: 하늘색 계열** (파란색 포인트)
  - 로고, 메인 배너, 검색창 테두리, 주요 버튼 등에 하늘색 적용
  - 흰색 배경 위 여백 있는 레이아웃
  - 시원하고 직관적인 인상

**참고 레이아웃**
- 상단: 로고 + 검색창 + 카테고리 네비게이션 + 우측 유틸리티(이벤트, 로그인 등) + [의뢰하기] 버튼
- 메인: 배너 캐러셀 (하늘색 계열 대형 배너)
- 필터: 카테고리, 스타일, 업종 드롭다운 + 정렬·찜 기능
- 메인 그리드: 좌측 카테고리 사이드바 + 서비스 카드 그리드(썸네일, 제목, 판매자명, 평점, 가격, 작업일, 찜 수)

---

## 📅 일정 & 예산

- **1차 MVP 기준** 견적 요청드립니다.
- 개발 기간: **약 4~6주** 예상
- 예상 예산 포함하여 견적·일정 안내 부탁드립니다.
