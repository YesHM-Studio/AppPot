## 2. DNS 레코드 관리 - 화면에서 할 일

지금 보시는 **"DNS 레코드 관리"** / **"호스트 등록"** 화면에서:

### ① 먼저: AppPot을 어디에 배포했나요?

| 배포 위치 | DNS 설정 방법 |
|-----------|---------------|
| **Render** | 아래 A안 참고 |
| **호스팅케이알 웹호스팅** | 아래 B안 참고 |
| **아직 미배포** | 먼저 Render 또는 호스팅에 배포 후 진행 |

---

### ②-A. Render에 배포했다면

1. [Render](https://render.com) 로그인 → AppPot 서비스 선택
2. **Settings** → **Custom Domains** → **Add Custom Domain**
3. `appot.kr` 입력 후 Render가 안내하는 **CNAME 주소** 또는 **IP** 확인

4. **호스팅케이알** 화면 → **DNS 레코드 관리** → **추가** 클릭 후:

| 유형 | 호스트 이름 | 값 |
|------|-------------|-----|
| CNAME | www | (Render에서 안내한 주소, 예: apppot-xxxx.onrender.com) |
| A | @ | (Render apex IP - Render가 제공하는 경우) |

※ Render가 apex(루트)용 A 레코드 IP를 주지 않으면, `www.appot.kr`만 먼저 연결하고 루트는 나중에 설정

---

### ②-B. 호스팅케이알 웹호스팅 사용한다면

1. 호스팅케이알 **웹호스팅** 관리에서 **서버 IP** 확인
2. **DNS 레코드 관리** → **추가** 클릭 후:

| 유형 | 호스트 이름 | 값 |
|------|-------------|-----|
| A | @ | (호스팅 웹서버 IP) |
| A | www | (같은 IP) |

---

### ③ 화면에서 클릭 순서

1. **"DNS 레코드 관리"** 영역에서 **추가** 또는 **호스트 등록** 버튼 클릭
2. **유형**: A 또는 CNAME 선택
3. **호스트 이름**: `@` (루트) 또는 `www` 입력
4. **값**: 서버 IP 또는 CNAME 주소 입력
5. **저장** 클릭

---

## 3. AppPot 배포 방식별 연결

### A. Render 배포 시
- Render 대시보드 → Settings → Custom Domains → `appot.kr`, `www.appot.kr` 추가
- Render가 안내하는 CNAME/A 레코드 값을 호스팅케이알 DNS에 등록

### B. 호스팅케이알 서버에 직접 배포 시
- 서버 IP를 DNS A 레코드에 등록
- 서버에 Node.js 앱을 PM2 등으로 실행 후 nginx/apache로 리버스 프록시 설정
